## useEffect

### 정의

useEffect는 외부 세계와 리액트를 **동기화**하는 훅입니다.

### 사용법

```tsx
const Component = () => {
  useEffect(() => {
    console.log('이펙트가 실행되었어요');

    return () => {
      console.log('클린업 함수가 실행되었어요');
    };
  }, []);
};
```
